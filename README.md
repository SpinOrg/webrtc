# webrtc [![Build Status](https://travis-ci.org/cubbit/webrtc.svg?branch=master)](https://travis-ci.org/cubbit/webrtc)

A CMake compliant webrtc build tool.

## How to build

```bash
make configure
make build-webrtc
```

## Debug

```bash
make configure-debug
make build-webrtc
```

## How to pack the artifacts for release

```bash
make pack
```
